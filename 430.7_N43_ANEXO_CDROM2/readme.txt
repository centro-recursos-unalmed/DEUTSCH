===============================================================================
1. Betriebssysteme
===============================================================================

Das Programm ist auf Windows- und auf Mac-Betriebssystemen nutzbar.

-------------------------------------------------------------------------------
a) Windows-Version
-------------------------------------------------------------------------------
Für die Windows-Version starten Sie bitte die Datei start.exe. Sie können dann 
wählen, ob Sie das Programm als eigene Anwendung oder als Browserversion in 
Ihrem Standardbrowser ansehen möchten.

-------------------------------------------------------------------------------
b) Mac-Version
-------------------------------------------------------------------------------
Für die Mac-Version starten Sie bitte die Datei index.html. Es öffnet sich die 
Browserversion in Ihrem Standardbrowser.


===============================================================================
2. Technische Voraussetzungen
===============================================================================
Um die Filme in der Windowsversion abspielen zu können, muss ein Video-Codec 
installiert sein. Ist der Codec auf Ihrem Rechner nicht vorhanden, so bietet das 
Intensivtraining die Installation des Codecs an. Für diese Installation benötigen 
Sie Administrationsrechte.
 
Für die Nutzung der Browserversion muss der Adobe Flash Player installiert sein. 
Es muss außerdem JavaScript aktiviert sein.
